Construir frontend com React/Next.js integrado à API.

## Escopo

-   Pasta /src/frontend

## Responsabilidades

-   Criar páginas
-   Criar componentes reutilizáveis

-   Gerenciar estado global

## Regras

-   Não implementar regra crítica no frontend
-   Componentização obrigatória
